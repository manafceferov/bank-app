İstifadəçilərin bank hesablarını, kartlarını, əmanətlərini və kredit ödənişlərini idarə edə biləcəyi, həm REST API, həm də Thymeleaf MVC dəstəyi olan rəqəmsal bankçılıq platforması.

Funksionallıq I

İstifadəçi idarəetməsi — Spring Security vasitəsilə təhlükəsiz qeydiyyat və giriş.

Hesab və Kart əməliyyatları — Yeni hesabların açılması, kartların idarə edilməsi və balansın izlənməsi.

Pul köçürmələri — Hesablar arası və ya daxili köçürmələrin sürətli və təhlükəsiz həyata keçirilməsi.

Kredit və Depozit idarəetməsi — Kredit müraciətləri, ödəniş cədvəlləri və depozitlərin monitorinqi.

Əməliyyat tarixçəsi — Bütün tranzaksiyaların və ödənişlərin detallı şəkildə izlənməsi.

Texnologiyalar
Java 21 — Backend biznes məntiqi və təbəqələr (Controller, Service, Repository) üçün.

Kotlin — Entity və DTO-ların yazılması, təmiz və qısa kod strukturu üçün (lombok istifadə etmək istəmədim).

Spring Boot — Layihənin əsas framework-ü.

Spring MVC + Thymeleaf — Dinamik və istifadəçi dostu veb interfeys.

Spring Security — İstifadəçi seanslarının təhlükəsizliyi və avtorizasiya.

PostgreSQL — Əsas əlaqəli məlumat bazası.

Liquibase — Verilənlər bazası miqrasiyalarının və versiyalaşdırılmasının idarə edilməsi.

MapStruct — Entity-lər və DTO-lar arası mapping prosesi üçün.

Docker — PostgreSQL və digər infrastruktur komponentlərinin konteynerləşdirilməsi.

Swagger UI — REST API endpoint-lərinin sənədləşdirilməsi və test edilməsi.
